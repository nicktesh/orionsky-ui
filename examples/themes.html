Theme System

OrionSky UI provides a comprehensive theme system with light and dark modes inspired by cosmic elements.

The theme system uses CSS variables to define colors, spacing, typography, and other design elements. This makes it easy to customize the appearance of your application.

OrionSky UI comes with two built-in themes:

  • Light Theme: A bright, clean design with subtle cosmic elements
  • Dark Theme: A deep space-inspired dark mode with enhanced cosmic elements

You can toggle between themes using the theme toggle button in the top-right corner of this page.

Using Theme Variables

You can use theme variables in your CSS to ensure your custom components adapt to theme changes:

.my-custom-element {
  background-color: var(--os-primary-color);
  color: var(--os-white-color);
  border-radius: var(--os-border-radius);
  box-shadow: var(--os-box-shadow);
}

Theme Switching

OrionSky UI provides JavaScript utilities for theme switching.

You can programmatically switch themes using the OrionSky UI JavaScript API:

// Initialize OrionSky UI
OrionSky.init();

// Set theme to dark mode
OrionSky.setTheme('dark');

// Set theme to light mode
OrionSky.setTheme('light');

// Toggle between light and dark mode
OrionSky.toggleTheme();

// Get current theme
const currentTheme = OrionSky.getTheme();
