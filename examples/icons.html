Icon System

OrionSky UI integrates with Phosphor Icons, a flexible icon family with a consistent style.

The icon system provides a simple way to use icons in your application with various styles, sizes, and colors.

OrionSky UI uses Phosphor Icons, which includes over 6,000 icons in various weights and styles.

Basic Usage

Learn how to use icons in your application.

HTML Attribute API

The simplest way to use icons is with the data attribute API:

<span data-os-icon="rocket"></span>
<span data-os-icon="star" data-os-icon-weight="fill"></span>
<span data-os-icon="planet" data-os-icon-size="lg" class="os-icon-primary"></span>

JavaScript API

You can also load icon weights programmatically:

// Load a specific icon weight
OrionSky.loadIconWeight('fill')
  .then(() => {
    console.log('Fill weight loaded');
  });
