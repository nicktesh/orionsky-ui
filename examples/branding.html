Logo Usage

Guidelines for using the OrionSky UI logo in your applications.

Basic Implementation

Use the appropriate logo version based on the background color:

<!-- Full logo on light background -->
<img src="../assets/branding/orionsky-logo.svg" alt="OrionSky UI" />

<!-- Full logo on dark background -->
<img src="../assets/branding/orionsky-logo-dark.svg" alt="OrionSky UI" />

<!-- Icon only on light background -->
<img src="../assets/branding/orionsky-icon.svg" alt="OrionSky UI" />

<!-- Icon only on dark background -->
<img src="../assets/branding/orionsky-icon-dark.svg" alt="OrionSky UI" />

Theme-Aware Implementation

To automatically switch between light and dark logos based on the theme:

<!-- Theme-aware full logo -->
<img src="../assets/branding/orionsky-logo.svg" alt="OrionSky UI" class="logo-light" />
<img src="../assets/branding/orionsky-logo-dark.svg" alt="OrionSky UI" class="logo-dark" />

<!-- CSS to show/hide based on theme -->
<style>
[data-theme="dark"] .logo-light {
  display: none;
}
[data-theme="light"] .logo-dark {
  display: none;
}
</style>

Logo Sizing

Recommended logo sizes:

  • Navbar/Header: 40px height
  • Sidebar: 30px height
  • Footer: 24px height
  • Minimum size for the full logo: 120px wide
  • Minimum size for the icon: 24px Ã— 24px

Always maintain the aspect ratio when resizing.

Typography

OrionSky UI uses the Inter font family as its primary typeface.

Inter Font Family

The Inter font is a variable font designed for computer screens. It features a tall x-height to aid in readability of mixed-case and lower-case text.
